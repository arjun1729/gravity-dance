# 🌌 Gravity Dance - Interactive N-Body Simulator

An interactive gravitational N-body simulator built with vanilla JavaScript. Visualize orbital mechanics, galaxy collisions, and chaotic gravitational systems in real-time.

![Gravity Dance](https://img.shields.io/badge/Physics-Simulation-blue)
![Interactive](https://img.shields.io/badge/Interactive-WebGL-green)
![Open Source](https://img.shields.io/badge/Open%20Source-MIT-orange)

## 🚀 Features

- **Real Physics**: Implements Newton's law of universal gravitation (F = G·m₁·m₂/r²)
- **Interactive**: Click and drag to create bodies with custom velocities
- **Beautiful Visualizations**: Particle trails, glowing effects, and smooth animations
- **Preset Scenarios**:
  - ☀️ Solar System - Classical planetary orbits
  - ⭐ Binary Stars - Two stars in mutual orbit with planetesimals
  - 🌀 Galaxy Collision - Two galaxies merging
  - 💫 Chaotic System - Random bodies in complex interactions

- **Adjustable Parameters**:
  - Time speed control (0.1x - 5x)
  - Toggle particle trails
  - Zoom and pan controls
  - Pause/resume simulation

## 🎮 Controls

| Action | Control |
|--------|---------|
| Add Body | Click and drag on canvas |
| Preset Scenarios | Click buttons in control panel |
| Time Speed | Adjust slider |
| Show/Hide Trails | Toggle checkbox |
| Pause/Resume | Press SPACEBAR |
| Zoom | Mouse wheel scroll |
| Clear All | Click "Clear All" button |
